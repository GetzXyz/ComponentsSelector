FAQ

What is FORGE AI PC Builder?+
FORGE is an AI-powered PC build recommendation tool designed for the international market. Enter your budget and use case (gaming or workstation), and FORGE uses GROQ API with live Google Search to recommend components at current market prices from sources like Daraz.pk, Amazon, eBay, OLX Pakistan and many other useful websites.
Are the prices accurate and up to date?+
FORGE uses Groq API with Google Search grounding to fetch current 2024–2025 prices from local and international retailers. However, prices fluctuate daily. Always verify prices directly with sellers before purchasing. FORGE is a research starting point, not a final price quote.
What currencies are supported?+
FORGE supports PKR, USD, EUR, GBP, AED, SAR, CAD, and AUD. Select your preferred currency from the dropdown — all prices are converted from PKR at approximate exchange rates. Use PKR for the most accurate Pakistani market prices.
What is the minimum budget?+
The minimum budget is Rs 30,000 PKR (or equivalent currency). At this level, FORGE recommends the most basic functional configuration. For a proper gaming or workstation build, we recommend at least Rs 80,000 PKR(or equivalent currency ).
Why does FORGE show "Using cached data"?+
This means the connection to Groq API failed due to some technical issues and FORGE is showing pre-built fallback recommendations instead. Common causes: (1) API key not configured or invalid, (2) internet connectivity issue, (3) API quota exceeded. The cached recommendations are still high-quality and budget-appropriate — they just may not reflect today's exact prices. Click "↺ Retry" to try again.
Can I trust Used / Refurbished recommendations?+
Used and refurbished options can offer excellent value, particularly for GPUs and CPUs. Always: (1) buy from sellers with good ratings and reviews, (2) test the component before finalizing payment, (3) verify the component's age and condition, (4) prefer sellers who offer a short return window.
Does FORGE guarantee component compatibility?+
No. FORGE makes compatibility-aware recommendations (e.g., matching CPU socket to motherboard platform), but you should always verify compatibility independently using tools like PCPartPicker.com before purchasing.
Is my data private?+
FORGE does not store your data on any server we operate. Your budget and use-case preference are sent to Groq API to generate recommendations. See our Privacy Policy for full details.
Why is there a "rate limit" error?+
FORGE limits API calls to 5 per minute to prevent accidental overuse of the Groq API quota. Simply wait 60 seconds and try again. If you need more frequent access, you can self-host FORGE with your own API key configured with a higher quota in Google AI Studio.
